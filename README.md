# ShoppyGlobe 

Welcome to **ShoppyGlobe**, a responsive and feature-rich e-commerce web application built using **React**, **Redux Toolkit**, **React Router**, and **Tailwind CSS**.

ðŸ”— **Live Demo / Repository**: [ShoppyGlobe.com GitHub](https://github.com/JainAadeesh/ShoppyGlobe.com)


##  Features

- Product listing with search
- Product details page
- Add/remove from cart
- Persistent cart with Redux
- Responsive UI with Tailwind CSS
- Code splitting via `React.lazy`

##  Tech Stack

- React
- Redux Toolkit
- Tailwind CSS
- React Router
- DummyJSON API
